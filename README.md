- **Wired ethernet cord and adapter**




# Software Utilized
- **ROS2 Jazzy Jalisco LTS**
- **UR Robot Driver**
- **Moveit2 and RViz**


# Project Goals
The goal of this project was to conduct a "pick and place" scenario using a series of waypoints, controlling the robotic arm in a scene with multiple objects to navigate and grasp an object from one tank and place it in another. I used a base of python for ROS2 node creation and collision object declaration, and supplementary YAML waypoints directory.
